\doxysubsection{Detailed Description}
Enum representing different authentication token statuses.

This enum defines statuses such as\+:
\begin{DoxyItemize}
\item VALID\+: Token is valid and can be used for authentication.
\item EXPIRED\+: Token has exceeded its lifetime and needs to be refreshed.
\item INVALID\+: Token is invalid due to tampering or other issues.
