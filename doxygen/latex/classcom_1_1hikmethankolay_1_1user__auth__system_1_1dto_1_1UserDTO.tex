\doxysubsection{Detailed Description}
Unified Data Transfer Object for all user operations. 

This class serves as a single DTO for handling user registration, updates, and display operations. It uses validation groups to apply different validation rules depending on the operation context\+:
\begin{DoxyItemize}
\item Registration.\+class\+: Strict validation for new user registration
\item Update.\+class\+: Lenient validation for user updates (optional fields)
\end{DoxyItemize}

The class includes security features such as write-\/only password serialization to prevent password leakage in API responses while allowing password input during registration and updates.
